uki.js

A minimal, d3-based Model-View framework for building larger systems out of linked bl.ock-esque views. Reasons why I rolled my own / its main selling points:

  • Zero build steps required; relies on native ES6 imports + distributed, lazy resource retrieval
  • No extra syntax / template language to learn; only requires knowledge of JS, HTML, and CSS
  • Mostly optional MVC philosophy—can be used or ignored
  • Streamlines the process of loading resources (e.g. data files, API calls, or view-specific stylesheets / HTML template files)
  • Processing-style (setup + draw) rendering pipeline that ensures that needed DOM elements and resources exist
  • Composable UI addon library includes standard components like tooltips, context menus, modals, and loading screens
  • Integrations for GoldenLayout and Vega; using them is often as simple as applying a Mixin
